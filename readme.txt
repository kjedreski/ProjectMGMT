# task biller 

- Desktop application built on electron stack

this app is used to track in real time how much time is being spent on a Task.

For instance:

<Task>        
<comment #1>           0:00
<comment #2>           0:00

Total Time spent on Task : <comment #1 time>  + <comment #2 time>




Refer to Documentation/DataStructure.txt as to how the data is currently being stored
